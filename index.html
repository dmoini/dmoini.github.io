Playroll

Playroll is an application that allows users to create playrolls, which replace the traditional idea of playlists. Using a set of unique algorithms, Playroll uses your provided music to create the perfect playlist. Worked on backend using Go, GORM, GraphQL, and PostgresDB.

Casper Programming Language

Collaborated with four other LMU students to create Casper: The Friendly Programming Language. Casper incorporates a speech-like, easy-to-use syntax that eliminates unnecessary verboseness from the programming experience. Casper is reminiscent of Python with nuanced features and syntax designed to make its meaning as transparent as possible and make programming an entirely pleasant affair.
